# 文岚书法管理系统

## 项目目标

解决书法工作室课时管理问题。

## MVP 范围

MVP 只做三件事，全部围绕「课时」闭环：


| 目标           | 用户价值               |
| ------------ | ------------------ |
| 老师 30 秒内完成签到 | 上课时不翻本子、不找名字，点选即完成 |
| 自动扣课时        | 签到成功即扣 1 课时，无需人工记账 |
| 查看剩余课时       | 随时能看到每位学生还剩多少课时    |


**MVP 边界：以上三项之外的功能，一律不进入 MVP。**

### MVP 不包含

- 请假、补课、续费
- 报表、统计、Dashboard
- 多老师、班级、权限
- 家长端、通知、导出

## 长期规划

### Phase2 — 运营扩展

- 学生请假登记
- 补课安排
- 续费提醒
- 签到历史查询

### Phase3 — 规模扩展

- 多老师协作
- 课消报表
- 家长课时查询
